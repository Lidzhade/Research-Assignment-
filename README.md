# Research Assignment 1 – Data Analytics Foundations

## Overview
This repository contains my structured breakdown and study approach for the Data Analytics Research Assignment 1. The goal was to build a strong conceptual foundation across key areas of data analytics, data engineering, and modern data systems.

## Study Structure

### Day 1: Data Fundamentals
Covered foundational data concepts including:
- Databases
- Data Warehouses
- Data Lakes
- Data Marts
- Data Types

### Day 2: Data Processing & Storage
Focused on core data engineering concepts:
- Data Modeling
- Data Mining
- DBMS (Database Management Systems)
- ETL (Extract, Transform, Load)
- Storage Formats

### Day 3: Artificial Intelligence & Advanced Analytics
Explored modern AI concepts:
- Artificial Intelligence (AI)
- Machine Learning (ML)
- Deep Learning
- Natural Language Processing (NLP)
- Large Language Models (LLMs)
- Generative AI
- Training Data
- Computer Vision

### Day 4: Tools & Technologies
Introduced key tools and platforms in data workflows:
- APIs
- JSON
- Prompt Engineering
- Cloud Computing
- Git & Version Control
- Jupyter Notebooks
- Python

### Day 5: Governance & Professional Practice
Covered responsible and professional data usage:
- Data Ethics
- Data Privacy
- Data Governance
- Career Paths in Data Analytics
- Personal Reflection

## Purpose of This Work
This exercise was completed to strengthen my understanding of end-to-end data concepts, from raw data storage to modern AI-driven applications, while building consistency in structured learning and documentation.

## Key Takeaway
Data analytics is not just about tools — it is about understanding the full lifecycle of data, from collection and storage to insight generation and responsible use.
